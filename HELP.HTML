JCalculate Help Document

Welcome to JCalculate, a versatile and user-friendly calculation tool designed to handle a wide range of mathematical operations. This document provides a comprehensive guide to the features and functionalities of JCalculate.

Core Features

  • Basic Arithmetic: Perform fundamental arithmetic operations, including addition (+), subtraction (-), multiplication (*), and division (/). These operations can be used in both Menu Mode and Command Line Mode.
  • Advanced Calculations: JCalculate provides a suite of advanced mathematical functions:
    • Power: Calculate the result of a number raised to a specified power using the Math.pow(base, exponent) function.
    • Square Root: Determine the square root of a number with the Math.sqrt(number) function.
    • Logarithm (base e): Compute the natural logarithm of a number using the Math.log(number) function.
    • Logarithm (base 10): Calculate the base-10 logarithm of a number with the Math.log10(number) function.
    • Trigonometric Functions: Calculate sine (Math.sin(angle)), cosine (Math.cos(angle)), and tangent (Math.tan(angle)) of an angle (in radians).
    • Inverse Trigonometric Functions: Find the arcsine (Math.asin(value)), arccosine (Math.acos(value)), and arctangent (Math.atan(value)) of a value.
  • Absolute Value, Ceiling, Floor, and Rounding:
    • Absolute Value: Obtain the non-negative value of a number using Math.abs(number).
    • Ceiling: Round a number up to the nearest integer with Math.ceil(number).
    • Floor: Round a number down to the nearest integer using Math.floor(number).
    • Rounding: Round a number to the nearest integer using Math.round(number).
  • Max and Min: Determine the largest (Math.max(number1, number2, ...)) or smallest (Math.min(number1, number2, ...)) number from a set of values.
  • Exponential Function: Calculate the exponential function (e raised to the power of a number) using Math.exp(number).

Modes of Operation

JCalculate offers two operational modes, providing flexibility for different user preferences and calculation complexities:

  • Menu Mode: This mode provides a user-friendly, point-and-click interface. Buttons are provided for each operation, and users can input numbers into the text field. This mode is ideal for simple calculations and users who prefer a visual interface.
  • Command Line Mode: In this mode, users enter mathematical expressions directly into the text field, using standard mathematical notation and function calls (e.g., "Math.sqrt(25) + 10"). This mode is more efficient for complex calculations, allowing users to combine multiple operations in a single expression. It is also suitable for users familiar with programming-style mathematical syntax.

You can easily switch between these modes using the "Menu" and "Command Line" buttons located in the application's interface.

Using JCalculate

Menu Mode

  1. Select Operation: Choose the desired mathematical operation (e.g., "Add," "Subtract," "Sqrt") by clicking the corresponding button in the application's button panel.
  2. Enter Values: Input the numerical values required for the operation into the text field. The application will guide you on how to input the values.
  3. Calculate: After entering the values, click the "Calculate" button to perform the operation. The result will be displayed in the output area.
  4. Clear: To reset the input field and output area, click the "Clear" button. This allows you to begin a new calculation.

Command Line Mode

  1. Enter Expression: Type a valid mathematical expression directly into the text field. Use standard mathematical notation and the provided function calls (e.g., "Math.pow(2, 3) + Math.sqrt(16)").
  2. Calculate: Click the "Calculate" button to evaluate the expression. The result will be shown in the output area.
  3. Clear: Use the "Clear" button to erase the contents of the input field and the output area, preparing the application for the next calculation.

Example: To calculate the square root of 25 and add 10 in Command Line Mode, you would enter "Math.sqrt(25) + 10" into the text field and then click "Calculate". The result, 15, would be displayed.

Error Handling

JCalculate incorporates robust error handling to ensure a smooth and reliable user experience. If an error occurs during a calculation (e.g., invalid input, division by zero), the application will display an informative error message in the output area, helping you understand the issue. Additionally, errors are logged to the "error_log.txt" file for debugging and troubleshooting purposes.
